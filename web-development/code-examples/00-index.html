Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating webpages.

HTML Tags and Elements

HTML is made up of elements, which are defined by tags. An element usually has an opening tag, content, and a closing tag:

        <p>This is how a paragraph is created with a paragraph element.</p>

Some examples of HTML tags include: <h1>, <p>, <a>, <div>, <img>, <ul>, <li>, <table>.
